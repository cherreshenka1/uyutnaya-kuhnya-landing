# Уютная кухня

Тестовое задание по вёрстке лендинга кафе

## Что внутри

- адаптивный лендинг под десктоп и `375px`;
- слайдер на 9 изображений со стрелками, точками и свайпом;
- аккордеон с FAQ;
- форма бронирования с базовой валидацией;
- модальное окно успешной отправки.

## Запуск

Можно открыть `index.html` напрямую в браузере или поднять простой локальный сервер:

```bash
python -m http.server 8000
```
